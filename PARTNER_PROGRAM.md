# Programme Partenaire TruTube - Guide Complet

## Philosophie

> **"Vous êtes partenaire, pas dépendant."**

TruTube considère les créateurs comme de véritables partenaires business, pas comme de simples utilisateurs à la merci d'algorithmes opaques.

## Conditions d'éligibilité

### 1. Identité vérifiée (KYC) - Obligatoire

**Vérifications requises** :
- ✅ Pièce d'identité officielle (carte nationale, passeport, permis)
- ✅ Email vérifié
- ✅ Téléphone vérifié
- ✅ Âge minimum 18 ans (ou représentant légal)
- ✅ Compte bancaire/Stripe validé

**Objectif** : Éviter fraudes, faux comptes, blanchiment

**Processus** :
1. Upload document d'identité
2. Selfie de vérification
3. Validation email (code)
4. Validation téléphone (SMS)
5. Connexion compte bancaire (Stripe)

**Délai** : 24-48h pour validation complète

### 2. Score d'authenticité ≥ 80/100

**Critères évalués** :
- Engagement réel (likes, commentaires, partages authentiques)
- Trafic non suspect (pas de pics anormaux)
- Pas d'achats de vues détectés
- Watch-time cohérent avec vues
- Diversité géographique naturelle
- Taux de rebond normal

**Score dynamique** : Recalculé en continu

**En dessous de 80** : Suspension automatique de la monétisation avec notification

### 3. Seuil minimum d'audience (au choix)

Le créateur doit atteindre **AU MOINS UN** des seuils suivants :

**Option A - Chaîne classique** :
- 1,000 abonnés
- ET 4,000 heures de visionnage sur 12 mois glissants

**Option B - Shorts/viraux** :
- 1,000,000 vues Shorts sur 90 jours glissants

**Positionnement** : Plus accessible que YouTube (même seuil mais transparence totale)

### 4. Minimum 3 vidéos publiques

- Contenu original ou sous licence
- Vidéos de qualité minimale (résolution, son)
- Pas de réuploads non autorisés

### 5. 30 jours d'activité réelle

- Compte créé depuis 30 jours minimum
- Activité régulière (uploads, connexions)
- Pas de compte dormant puis soudainement actif

### 6. Zéro violation majeure

**Violations bloquantes** :
- Contenus illégaux
- Incitation à la haine
- Désinformation volontaire grave
- Exploitation de mineurs
- Contenu adulte non conforme
- Violation répétée droits d'auteur

**Une seule violation majeure** = Exclusion du programme

### 7. Droits sur le contenu

Le créateur certifie :
- Posséder tous les droits sur ses contenus
- OU détenir licences écrites valides
- Musiques, images, vidéos tiers : autorisations obligatoires

**Réuploads non autorisés** : Exclus de la monétisation

## Types de monétisation

### 1. Publicité classique (65% créateur / 35% plateforme)

**Fonctionnement** :
- Ads pre-roll (avant la vidéo)
- Ads mid-roll optionnels (créateur décide)
- Bannières non intrusives

**Contrôle créateur** :
- Peut refuser certaines catégories (alcool, jeux d'argent, etc.)
- Peut désactiver mid-roll
- Peut blacklist certaines marques

**Estimation** : 2-5€ pour 1000 vues (CPM moyen Europe)

### 2. Abonnements créateur (80% / 20%)

**Paliers personnalisables** :
- Silver : 4.99€/mois
- Gold : 9.99€/mois
- VIP : 19.99€/mois

**Avantages créateur** :
- Contenu exclusif pour abonnés
- Lives privés
- Badges personnalisés
- Communauté fermée
- Emojis custom

**Estimation** : 100 abonnés Silver = 400€/mois net

### 3. Tips instantanés (95% / 5%)

**Fonctionnement** :
- Bouton "Support" sous chaque vidéo
- Paiement immédiat (Stripe)
- Visible en live pour encourager
- Montants libres : 2€, 5€, 10€, 20€, custom

**Avantages** :
- Commission minimale (5% seulement)
- Paiement instantané
- Pas de seuil minimum

**Estimation** : Variables, mais 50-200€/mois pour créateurs moyens

### 4. Marketplace services (85% / 15%)

**Services proposables** :
- Montage vidéo pour autres créateurs
- Design miniatures/bannières
- Community management
- Consulting stratégie contenu
- Formations privées

**Avantages** :
- Tarifs libres
- Paiement sécurisé
- Système d'avis

**Estimation** : 500-2000€/mois pour prestataires actifs

### 5. Sponsoring direct (90% / 10%)

**Fonctionnement** :
- Marques proposent sponsoring via TruTube Studio
- Négociation transparente
- Contrats générés automatiquement
- Paiement garanti

**Avantages** :
- Pas besoin d'agent
- Sécurité juridique
- Commission faible

**Estimation** : 500-5000€ par sponsoring selon audience

### 6. Collaborations (100% répartition libre)

**Fonctionnement** :
- Créateurs définissent % entre eux
- Ex : 50/50, 60/40, 70/30
- Contrat interne TruTube
- Revenus partagés automatiquement

**Avantages** :
- Aucune commission plateforme
- Transparence totale
- Simplicité administrative

## Suspension & appels

### Motifs de suspension

1. **Activité suspecte détectée**
   - Pic de vues anormal
   - Achats de bots détectés
   - Comportement suspect

2. **Violation grave des règles**
   - Contenu interdit
   - Harcèlement
   - Spam

3. **Manipulation d'engagement**
   - Bots
   - Achats de likes/commentaires
   - Fraude clic publicitaire

4. **Fraude publicitaire**
   - Clic frauduleux sur publicités
   - Génération artificielle de vues

5. **Violation droits d'auteur répétée**
   - 3 strikes copyright
   - Réuploads massifs

### Processus de suspension

**Étape 1 - Notification** :
- Email + notification in-app
- Raison détaillée
- Preuves fournies
- Délai : **48h maximum** après détection

**Étape 2 - Examen créateur** :
- Créateur dispose de **7 jours** pour répondre
- Peut soumettre contre-arguments
- Peut demander révision humaine

**Étape 3 - Décision finale** :
- Révision par équipe humaine
- Décision sous **14 jours**
- Notification avec justification complète

### Droit d'appel

**Système d'appel en 3 niveaux** :

1. **Premier niveau** : Équipe modération
2. **Deuxième niveau** : Manager senior
3. **Troisième niveau** : Comité indépendant créateurs

**Garanties** :
- Tout créateur peut faire appel
- Processus gratuit
- Délai maximum traitement : 30 jours
- Transparence totale des décisions
